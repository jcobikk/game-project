------------------------
OPIS GRY
------------------------
„Zgadnij liczbę” to prosta gra konsolowa, w której gracz musi odgadnąć
losowo wybraną liczbę z określonego zakresu. Program informuje, czy
podana liczba jest za mała, za duża lub poprawna.

------------------------
URUCHOMIENIE
------------------------
1. Uruchomić plik .exe

------------------------
STEROWANIE
------------------------
- Gra obsługiwana jest z klawiatury
- Wpisuj liczby i zatwierdzaj klawiszem ENTER
- Program na bieżąco reaguje na wprowadzone dane

------------------------
MOŻLIWOŚCI / FUNKCJE
------------------------
- Losowanie liczby przy każdym uruchomieniu gry
- Informowanie gracza, czy liczba jest:
  * za mała
  * za duża
  * poprawna
- Liczenie liczby prób
- Prosta i czytelna obsługa w konsoli

------------------------
WYMAGANIA
------------------------
- System operacyjny: Windows / Linux / macOS
- Konsola / terminal
